# 06 - Counter Demo

The canonical Flutter template application highlighting modern project structures, theme styling inheritance, and floating action button handlers.

## 🚀 Key Concepts Covered
- **Flutter Project Structure:** Standard scaffolding generated by the CLI, detailing entry hooks and default configs.
- **Theme Data inheritance:** Setting seed colors (`ColorScheme.fromSeed`) and consuming the active context theme dynamically (`Theme.of(context).colorScheme.inversePrimary`).
- **Floating Action Buttons (FAB):** Placing primary interactions floating dynamically over screens.

## 🛠️ Widgets Used
- `FloatingActionButton` - Placed inside the Scaffold's designated FAB slot.
- `ThemeData` & `ColorScheme` - Dynamically dictates application styling.
- `Column` - Arranges child text elements vertically centered.

## 🏃 Run this Lab
To run this specific lab:
1. Navigate to this directory:
   ```bash
   cd 06_counter_demo
   ```
2. Fetch dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
