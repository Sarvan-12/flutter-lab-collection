# 01 - Hello World

A minimal Flutter application that serves as the entry point to mobile development. It displays a centered text greeting on the screen.

## 🚀 Key Concepts Covered
- **Flutter App Lifecycle:** Basic entry point of a Flutter application (`main()` and `runApp()`).
- **Stateless Widgets:** Using `StatelessWidget` for static UI presentation.
- **Material Design structure:** Implementing `MaterialApp`, `Scaffold`, and `AppBar`.

## 🛠️ Widgets Used
- `MaterialApp` - Configures the application design system.
- `Scaffold` - Provides the default visual structure (app bar, body).
- `AppBar` - Header widget with centered title.
- `Center` - Aligning elements perfectly in the middle.
- `Text` & `TextStyle` - Rendering styled text with custom font size and weight.

## 🏃 Run this Lab
To run this specific lab:
1. Navigate to this directory:
   ```bash
   cd 01_hello_world
   ```
2. Fetch dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
