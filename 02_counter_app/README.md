# 02 - Counter App

A simple interactive application demonstrating user interaction and dynamic state updates in Flutter using `StatefulWidget` and `setState`.

## 🚀 Key Concepts Covered
- **Stateful Widgets:** Difference between `StatelessWidget` and `StatefulWidget`.
- **State Management:** Mutating state and rebuilding the UI using `setState()`.
- **Layouts:** Arranging widgets vertically (`Column`) and horizontally (`Row`).

## 🛠️ Widgets Used
- `StatefulWidget` - Widget that maintains mutable state.
- `Column` - Vertically aligns the counter text and control buttons.
- `Row` - Horizontally aligns the increment (`+`) and decrement (`-`) buttons.
- `ElevatedButton` - Material design buttons that trigger state changes.
- `SizedBox` - Adds structural spacing between layouts.
