# 03 - Login UI

A clean form interface demonstrating user input handling, authentication simulation, and visual snackbar feedback in Flutter.

## 🚀 Key Concepts Covered
- **Form Input Handling:** Capturing text inputs using `TextEditingController`.
- **Obscuring Text:** Hiding sensitive inputs (passwords) inside fields.
- **User Feedback:** Showing temporary overlay messages using `SnackBar`.

## 🛠️ Widgets Used
- `TextField` - Receives user typing input, utilizing `decoration` for labels.
- `TextEditingController` - Listens to and controls form fields programmatically.
- `ScaffoldMessenger` & `SnackBar` - Displays a slide-up notification from the bottom.
- `Padding` - Introduces whitespace margins around the form fields.
