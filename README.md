## Clone & Setup

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed and configured
- An active emulator or connected physical device

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/Sarvan-12/flutter-lab-collection.git

# 2. Navigate into any lab folder
cd flutter-lab-collection/01_hello_world

# 3. Fetch dependencies
flutter pub get

# 4. Run the app
flutter run
```

> **Note:** Each directory is a self-contained Flutter project. Execute steps 2–4 inside whichever lab folder you wish to run.
