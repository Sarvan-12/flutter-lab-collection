## 🚀 Clone & Setup

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed and on your PATH
- A connected device or emulator

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/Sarvan-12/flutter-lab-collection-.git

# 2. Navigate into any lab folder
cd flutter-lab-collection-/01_hello_world

# 3. Fetch dependencies
flutter pub get

# 4. Run the app
flutter run
```

> Each lab is an independent Flutter project. Repeat steps 2–4 for any lab you want to run.
